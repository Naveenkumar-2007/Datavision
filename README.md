# DataVision - AI Business Analyst

An autonomous AI-powered data analysis platform with:
- 🧠 **Autonomous Brain** - Intelligent query understanding
- 📊 **Smart Visualizations** - Auto-generated charts
- 🤖 **AutoML** - Automated machine learning
- 🔮 **Predictive Analytics** - Forecasting & anomaly detection
- 📧 **Email Reports** - Scheduled insights delivery

## Getting Started

1. Upload your CSV/Excel data
2. Ask questions in natural language
3. Get AI-powered insights and visualizations

## Required Secrets

Configure these in Space Settings → Secrets:
- `GROQ_API_KEY` - Groq API key for LLM
- `SUPABASE_URL` - Supabase project URL
- `SUPABASE_ANON_KEY` - Supabase anonymous key
- `RESEND_API_KEY` - Resend email API key (optional)
