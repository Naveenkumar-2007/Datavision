# Data Vision - Universal Analytics Platform

An intelligent universal analytics platform powered by AI. Works with any dataset - HR, Sales, Finance, Inventory, and more.

## Features
- 🤖 AI-powered chat analyst with 7 analysis modes
- 📊 Universal visualization (18+ chart types)
- 📈 Automated reports and insights
- 📁 Dynamic data upload and analysis
- 🕸️ Knowledge graphs and mindmaps
- 🔮 Predictive analytics
- 🔔 Email notifications

## Key Capabilities
- **RAG Mode**: Document-based Q&A with charts
- **GraphRAG Mode**: Relationship visualization
- **Hybrid Mode**: Combined intelligence
- **Vision Mode**: Image analysis
- **Prediction Mode**: Forecasting with confidence intervals
- **Agentic RAG**: Multi-step reasoning
- **Multi-RAG**: Cross-document analysis

## Tech Stack
- **Frontend**: React + TypeScript + Tailwind CSS
- **Backend**: FastAPI + Python
- **Database**: Supabase (PostgreSQL)
- **AI**: Multiple LLM support (Groq, OpenRouter)
- **Email**: Resend
